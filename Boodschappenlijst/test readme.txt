1. installeer node.js
2. open cmd en ga naar Boodschappenlijst directory (daar waar dit bestaand staat)
3. typ: npm install

Om Karma Tests te runnen:
4. typ: karma start karma.conf.js

Om Protractor E2E Tests te runnen:
5. typ: webdriver-manager update
6. typ: webdriver-manager start
7. open nieuwe cmd terminal
8. in nieuwe terminal ga naar Boodschappenlijst directory (daar waar dit bestaand staat)
9. typ daar: protractor protractor.conf.js
